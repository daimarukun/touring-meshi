Google Maps APIキーの設定

このアプリは Google の地図・店舗データを使います。ご自身のAPIキーを入力してください（ブラウザ内にのみ保存され、外部には送信しません）。

  1. Google Cloud Console でプロジェクトを作成
  2. 「Maps JavaScript API」と「Places API (New)」を有効化
  3. 「認証情報」→「APIキーを作成」でキーを取得
  4. （推奨）キーに「HTTPリファラー」制限をかけて悪用を防止
